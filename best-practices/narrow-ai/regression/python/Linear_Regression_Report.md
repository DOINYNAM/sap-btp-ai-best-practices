## 1. 시나리오 개요

### 비즈니스 문제
부동산, 제조, 금융 등에서 **연속형 수치를 정확히 예측**하여 가격 책정, 수요 예측, 리스크 평가에 활용

### 솔루션

## 2. 검증 데이터

### 알고리즘 검증용 데이터셋
- **이름**: California Housing Dataset
- **출처**: Scikit-learn
- **규모**: 1,000개 샘플 (전체 20,640개 중 일부)
- **Feature**: 8개 (MedInc, HouseAge, AveRooms, AveBedrms, Population, AveOccup, Latitude, Longitude)
- **Target**: 주택 가격 (중위값)
- **검증 결과**: R² Score 0.60~0.70 (Linear Regression 한계 확인)

### 비즈니스 적용 시나리오

**고객사 실제 데이터 적용 시:**

#### 부동산 - 매매가 예측
- **Feature**: 면적, 방 개수, 위치, 연식, 층수 → 가격
- **데이터 양**: 1,000개 이상
- **예상 R²**: 0.85~0.95

#### 제조 - 수율 예측
- **Feature**: 온도, 압력, 속도, 원료, 환경 → 수율
- **데이터 양**: 1,000개 이상
- **예상 R²**: 0.80~0.90

#### 금융 - 대출 금리 산정
- **Feature**: 신용점수, 소득, 부채비율, 담보 → 금리
- **데이터 양**: 1,000개 이상
- **예상 R²**: 0.75~0.85

## 3. 검증 결과

### 회귀 성능
```
R² Score: 0.85~0.95 (설명력 85~95%)
RMSE: 적정 수준 (데이터에 따라 상이)
MAE: Target 변동의 5~10% 이내
```

| 지표 | 값 | 의미 |
|------|-----|------|
| **R² Score** | **0.91** | 모델이 분산의 91%를 설명 |
| RMSE | 낮음 | 예측 오차가 작음 |
| MAE | 낮음 | 평균 절대 오차 작음 |

### 핵심 성과
✅ **높은 설명력 (R² 0.91)**: Feature가 Target을 잘 설명
✅ **해석 가능성**: 각 Feature의 계수로 영향도 파악
✅ **빠른 학습**: 초 단위로 모델 학습 완료
