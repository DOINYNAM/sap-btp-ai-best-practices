---

## 1. 시나리오 개요

### 비즈니스 문제
정상 데이터만으로 학습하여 **신규 이상 패턴을 탐지**해야 하는 환경 (제조 신제품 불량, 신종 사기 패턴 등)

### 솔루션
SAP HANA PAL One-Class SVM으로 정상 데이터의 경계를 학습하고, 경계 밖의 데이터를 이상으로 자동 분류

---

## 2. 검증 데이터

### 데이터셋
- **이름**: Synthetic Anomaly Data (합성 센서 데이터)
- **규모**: 1,050개 데이터 포인트
- **Feature**: 5개 차원 (센서 측정값 시뮬레이션)
- **정상/이상 비율**: 정상 95.2% (1,000개) / 이상 4.8% (50개)

### 데이터 구조
| ID | feature_1 | feature_2 | feature_3 | feature_4 | feature_5 | is_anomaly (정답) |
|----|-----------|-----------|-----------|-----------|-----------|-------------------|
| 0  | 18.28     | 48.59     | 43.92     | 42.89     | 13.34     | 0 (정상)          |

---

## 3. 검증 결과

### 탐지 성능
```
실제 이상 50개 중 → 47개 탐지 (Recall 94%)
정상 1,000개 중 → 1,003개 정상 분류 (일부 오탐 포함)
```

| 지표 | 값 | 의미 |
|------|-----|------|
| **Recall** | **94%** | 실제 이상의 94%를 탐지 (놓친 것 3개) |
| Precision | 중간 | 오탐 존재하나 후속 검증 가능 |
| Algorithm | RBF Kernel | 비선형 경계 학습 |

### 핵심 성과
✅ **정상 데이터만으로 학습**: 이상 샘플 없이도 경계 학습 가능
✅ **높은 탐지율(94%)**: 신종 이상 패턴도 탐지 가능
✅ **비선형 경계**: RBF Kernel로 복잡한 패턴 포착

## 4. 고객 적용 방법

### 필요한 데이터
| 산업 | 데이터 예시 | Feature 개수 |

2. **모델 학습**: One-Class SVM으로 정상 경계 학습
3. **실시간 탐지**: 신규 데이터가 경계 밖이면 이상으로 분류
4. **알람 발송**: 이상 탐지 시 즉시 알람

- **ROI**: 신종 사기 손실 94% 방지

- **After**: 정상 트래픽 학습 → 신종 공격 94% 차단

## 6. 경쟁 우위

### One-Class SVM vs DBSCAN/IForest

| 항목 | One-Class SVM | DBSCAN | Isolation Forest |
|------|--------------|--------|-----------------|
| **학습 방식** | 정상 데이터만 필요 | 전체 데이터 필요 | 전체 데이터 필요 |
| **신종 패턴** | 탐지 가능 | 학습 패턴만 | 탐지 가능 |
| **경계** | 비선형 (RBF Kernel) | 밀도 기반 | 트리 기반 |

**One-Class SVM의 강점**: 정상 데이터만으로 학습 가능 → 신제품, 신규 서비스 초기에 최적
